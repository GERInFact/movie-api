myFlix

This API provides an access to information about different movies, directors and genres. As a user you can sign up, update your personal information and create a list of your favorite movies.

  • Endpoint URL: '/'
    Method: GET
    Parameters: none

    Returns a welcome sentence.

    Response: Text
    Request Body: none

  • Endpoint URL: '/movies'
    Method: GET
    Parameters: none

    Returns a list of all movies.

    Response: JSON (array)
    Request Body: none

  • Endpoint URL: 'movies/featured'
    Method: GET
    Parameters: none

    Returns a list of all featured movies.

    Response: JSON (array)
    Request body: none

  • Endpoint URL: '/movies/:title'
    Method: GET
    Parameters: :title (required)

    Returns movie information based on its title.

    Response: JSON
    Request Body: none

  • Endpoint URL: '/movies/:title/genre'
    Method: GET
    Parameters: :title (required)

    Returns the genre of a specific movie, selected by its title

    Response: JSON
    Request body: none

  • Endpoint URL: '/movies/:title/director'
    Method: GET
    Parameters: :title (required)

    Returns director information of the movie selected by its title

    Response: JSON
    Request body: none

  • Endpoint URL: '/users'
    Method: POST
    Parameters: none

    Adds a new user to the database and returns the submitted data, if successful

    Response: Text
    Request body: JSON
    Format: {"username" : "value","password" : "value", "email": "value", "dateOfBirth" : "value" }

  • Endpoint URL: '/users/:username'
    Method: PUT
    Parameters: :username (required)

    Updates the user data based on the information sent to the server

    Response: Text
    Request body: JSON
    Format: {"username" : "value","password" : "value", "email": "value", "dateOfBirth" : "value" }

  • Endpoint URL: '/users/:username/:movies/:title'
    Method: DELETE
    Parameters: :username (required), :movies (required), :title (required)

    Deletes a specific movie, selected by its title, from the users movie list

    Response: Text
    Request body: none

  • Endpoint URL: '/users/:username'
    Method: DELETE
    Parameters: :username (required)

    Deletes a specific user from the database, based on the username submitted

    Response: Text
    Request body: none

  • Info

    Replace ':parameter' in your URL with some value

    URL: http://localhost:8080/movies/Snowden

    Response (JSON): { "title": "Snowden", "director": "Oliver Stone", "releaseDate": "2019-08-30T10:30:55.034Z", "views": 183.3791607679649, "rating": 3.282957315261641, "description": "An epic movie.", "genre": "Biographical Thriller", "imgUrl": "https://en.wikipedia.org/wiki/Snowden_(film)#/media/File:Snowden_film_poster.jpg" }

    All request are resolved at port: 8080
