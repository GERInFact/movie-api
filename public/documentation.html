myFlix

This API provides an access to information about different movies, directors and genres. As a user you can sign up, update your personal information and create a list of your favorite movies.

  • Endpoint URL: '/'
    Method: GET
    Parameters: none

    Returns a welcome sentence.

    Response: Text
    Request Body: none

  • Endpoint URL: '/movies'
    Method: GET
    Parameters: none

    Returns a list of all movies.

    Response: JSON (array)
    Request Body: none

  • Endpoint URL: 'movies/featured'
    Method: GET
    Parameters: none

    Returns a list of all featured movies.

    Response: JSON (array)
    Request body: none

  • Endpoint URL: '/movies/:title'
    Method: GET
    Parameters: :title (required)

    Returns movie information based on its title.

    Response: JSON
    Request Body: none

  • Endpoint URL: '/movies/:title/genre'
    Method: GET
    Parameters: :title (required)

    Returns the genre of a specific movie, selected by its title

    Response: JSON
    Request body: none

  • Endpoint URL: '/movies/genres/:genre'
    Method: GET
    Parameters: :genre (required)

    Returns a list of movies of a certain genre

    Response: JSON
    Request body: none

  • Endpoint URL: '/movies/:title/director'
    Method: GET
    Parameters: :title (required)

    Returns the director of the movie selected by its title

    Response: JSON
    Request body: none

  • Endpoint URL: '/users'
    Method: GET
    Parameters: none

    Returns a list of all users

    Response: JSON
    Request body: none

  • Endpoint URL: '/users/:username'
    Method: GET
    Parameters: :username (required)

    Returns a list of a certain user selected by his username

    Response: JSON
    Request body: none

  • Endpoint URL: '/users'
    Method: POST
    Parameters: none

    Adds a new user to the database and returns the submitted data, if successful

    Response: Text
    Request body: JSON
    Format: {"username" : String,"password" : String, "email": String, "dateOfBirth" : String / Date }

  • Endpoint URL: '/users/:username'
    Method: PUT
    Parameters: :username (required)

    Updates the user data based on the information sent to the server

    Response: Text
    Request body: JSON
    Format: {"username" : String, "password" : String, "email": String, "dateOfBirth" : String / Date }

  • Endpoint URL: '/users/:username/movies/:movieId'
    Method: PUT
    Parameters: :username (required), :moveiId (required)

    Updates the favorite movies of a certain user

    Response: JSON
    Request body: none
    Format:none

  • Endpoint URL: '/users/:username/movies/:movieId'
    Method: DELETE
    Parameters: :username (required), :movieId (required)

    Deletes a specific movie, selected by its id, from the users favorite movies

    Response: JSON
    Request body: none

  • Endpoint URL: '/users/:username'
    Method: DELETE
    Parameters: :username (required)

    Deletes a specific user from the database, based on the username submitted

    Response: Text
    Request body: none

  • Info

    Replace all ':parameter' in your URL with proper values

    E.g. http://localhost:8080/movies/Silence of the Lambs

    Response (JSON): { "Genre": { "Name": "Thriller", "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involves excitement and suspense in the audience." }, "Director": { "Name": "Jonathan Demme", "Bio": "Updated Bio", "Birth": "1944", "Death": "2017" }, "Actors": [], "_id": "5d734738dc0f0a45d0d2d917", "Title": "Silence of the Lambs", "Description": "Updated at the night", "ImagePath": "silenceofthelambs.png", "Featured": true }

    All requests are resolved at port: 8080
